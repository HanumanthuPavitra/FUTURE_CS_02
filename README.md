# Phishing Detection & Awareness Report

## Cyber Security Task 2 – Future Interns

### Objective

To analyze phishing email samples, identify phishing indicators, classify email risk, and develop user awareness recommendations.

### Email Samples

- MY Bharat Internship Email
- Innovexis Meeting Reminder
- Account Verification Phishing Email

### Tools Used

- Public Email Samples
- Browser Tools
- Google Message Header Analyzer
- Canva

### Deliverables

- Phishing Detection Report
- Risk Classification
- Awareness Guidelines
- Supporting Evidence

### Disclaimer

All email samples were analyzed for educational purposes only. No malicious links were accessed and no credentials were submitted.
